What is Conway's Game of Life?

Conway's Game of Life, otherwise known simply as Life, is a cellular automaton devised by the British mathmatician John Horton Conway.


Rules

  1. Any live cell with fewer than two live neighbours dies, as if caused by underpopulation.
  2. Any live cell with two or three live neighbours lives on to the next generation.
  3. Any live cell with more than three live neighbours dies, as if by overpopulation.
  4. Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

Basic Controls

Left Click: Switch cell state

Step Button: Create next generation

Step Speed: Automatic generation creation

Presets: Click to load preset


How does it work?

The intial pattern you create sets the seed of the system. The first generation is created through applying the four rules above to every cell. Births and deaths occur simultaneously. These rules are continually applied repeatedly to create further generations.
